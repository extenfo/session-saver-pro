Session Saver Pro — Privacy Policy

Last updated: 2026-01-02

Session Saver Pro is a tab & window session manager. It lets you save your current windows/tabs as a session, search sessions by URL/title/domain, and restore sessions into new windows.

Data we collect

  • We do not collect personal data on a server.
  • All session data and settings are stored locally in your browser using chrome.storage.local.

Data we share

  • We do not sell, share, or transmit your data to third parties.
  • No external analytics or tracking SDKs are used.

Permissions justification

  • storage — store saved sessions and settings locally.
  • tabs — read open tabs (URL/title) to save/restore sessions.
  • windows — manage browser windows to restore saved sessions and reopen them as requested by the user.
  • alarms — schedule autosave at a user-defined interval.
  • <all_urls> host access — required to save/restore tabs for any website the user has open.

Remote code

  • The extension does not download or execute remote code. All executable code is packaged with the extension.

Contact

If you have questions, contact us via the email provided on the Chrome Web Store listing.
